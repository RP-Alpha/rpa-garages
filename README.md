# rpa-garages

Public and private garage management system.

## Features
- **Storage**: Store and retrieve owned vehicles.
- **Valet Peds**: Interact with valet peds via Target to access garage.
- **Key Integration**: Automatically gives keys via `rpa-vehiclekeys` on spawn.

## Installation
1. Ensure `rpa-lib` and `rpa-vehiclekeys` are started.
2. Add `ensure rpa-garages` to your `server.cfg`.
